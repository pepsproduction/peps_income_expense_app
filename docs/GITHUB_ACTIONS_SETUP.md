# วิธีการเชื่อมต่อ GitHub กับ Google Apps Script (ใช้งานจากที่ไหนก็ได้)

เนื่องจาก Google Apps Script (GAS) ไม่มีปุ่มเชื่อมต่อกับ GitHub โดยตรงจากหน้าเว็บ (เนื่องจากเป็นระบบปิดของ Google) เราจึงใช้ **GitHub Actions (CI/CD)** มาทำหน้าที่เป็นตัวกลางในการนำส่งโค้ดจาก GitHub ไปยัง Google Apps Script โดยอัตโนมัติ

วิธีนี้จะช่วยให้คุณสามารถแก้ไขโค้ดผ่าน **GitHub จากหน้าเว็บหรือจากคอมพิวเตอร์เครื่องใดก็ได้ในโลก** และเมื่อคุณกดเซฟ (Commit & Push) ระบบจะทำการดึงโค้ดนั้นไปอัปเดตลง Google Apps Script ให้คุณทันทีโดยที่คุณไม่ต้องทำอะไรเพิ่ม!

---

## ขั้นตอนการตั้งค่า (ทำครั้งเดียว)

เพื่อให้ GitHub มีสิทธิ์ส่งโค้ดเข้าไปยัง Google Apps Script แทนเรา เราจำเป็นต้องนำสิทธิ์การเข้าถึง (Credentials) ไปบันทึกไว้ใน GitHub Secrets ตามขั้นตอนดังนี้:

### ขั้นตอนที่ 1: คัดลอกสิทธิ์การใช้งานจากเครื่องของคุณ
1. เปิดเครื่องคอมพิวเตอร์ของคุณที่เคยทำ `clasp login` สำเร็จไว้
2. เข้าไปที่โฟลเดอร์ผู้ใช้ของคุณ เพื่อหาไฟล์ชื่อ `.clasprc.json` 
   * ไฟล์นี้จะอยู่ที่ตำแหน่ง: **`C:\Users\bank_\.clasprc.json`**
3. ดับเบิลคลิกเปิดไฟล์ด้วย Notepad หรือ Text Editor ใดๆ 
4. **คัดลอก (Copy) ข้อความทั้งหมด** ภายในไฟล์นี้ไว้ (ข้อมูลจะเป็น JSON ยาวๆ ที่มี Token เข้าถึงระบบ Google ของคุณ)

### ขั้นตอนที่ 2: นำสิทธิ์ไปใส่ใน GitHub Secrets
1. เปิดบราวเซอร์และเข้าไปที่ GitHub Repository ของคุณ:
   [https://github.com/pepsproduction/peps_income_expense_app](https://github.com/pepsproduction/peps_income_expense_app)
2. คลิกที่แถบเมนู **Settings** (รูปฟันเฟืองด้านขวาบน)
3. ที่เมนูด้านซ้าย ให้เลื่อนลงไปที่หัวข้อ **Security** -> คลิกเลือก **Secrets and variables** -> เลือก **Actions**
4. คลิกปุ่มสีเขียว **New repository secret**
5. ตั้งค่าดังนี้:
   * **Name**: `CLASPRC_JSON`
   * **Secret**: (วางข้อความ JSON ทั้งหมดที่คุณคัดลอกมาจากไฟล์ `.clasprc.json` ลงไป)
6. คลิกปุ่ม **Add secret** เพื่อบันทึก

## วิธีการใช้งานหลังจากตั้งค่าเสร็จแล้ว

เมื่อคุณตั้งค่าเสร็จแล้ว คุณสามารถทำงานจาก **เครื่องไหนก็ได้** ดังนี้:

### 1. วิธีแก้ไขโค้ดและอัปเดตแอป (จากคอมพิวเตอร์เครื่องใดก็ได้)
* **วิธีที่ 1 (ผ่านหน้าเว็บ):** ล็อกอินเข้า GitHub.com -> เข้าไปแก้โค้ดในโฟลเดอร์ `apps-script/` บนบราวเซอร์ได้โดยตรง -> กด Commit เพื่อเซฟ
* **วิธีที่ 2 (ผ่านคอมเครื่องอื่นๆ):** Clone โปรเจกต์ลงเครื่องอื่น -> แก้ไขโค้ด -> สั่ง `git push` กลับมาที่ GitHub
* **หลังจากเซฟโค้ด:** GitHub Actions จะทำงานหลังบ้านโดยอัตโนมัติ (ใช้เวลาประมาณ 1 นาที) เพื่อนำโค้ดล่าสุดไป Deploy ใน Google Apps Script ของคุณทันที! คุณสามารถคลิกดูสถานะการอัปเดตได้ที่แท็บ **Actions** บน GitHub

> [!IMPORTANT]
> ทุกครั้งที่ระบบอัปเดตโค้ดไปยัง Google Apps Script อัตโนมัติแล้ว คุณอาจจำเป็นต้องเข้าไปที่ Google Apps Script Editor แล้วกด **Deploy > Manage Deployments** จากนั้นเลือกสร้างเป็น **Version ใหม่** (กรณีที่ใช้ลิงก์ Deploy ในการรันงาน) แต่ถ้าใช้เวอร์ชันทดสอบ (Dev link) โค้ดจะอัปเดตและทำงานทันที

### 2. วิธีการใช้งานแอปพลิเคชัน (สำหรับผู้ใช้งานทั่วไป)
* คุณมีลิงก์ Web App ของ Google Apps Script ที่เราทดสอบไปแล้วอยู่แล้ว:
  [ลิงก์ Web App ของคุณ](https://script.google.com/macros/s/AKfycbx5yYgsPSum9eSPC316wKYylhCOWSONb5DWAyZ-8g7ut9F7p3i3_0k9uq3lVyr1E05M/exec)
* ลิงก์นี้สามารถเปิดใช้งานได้จาก **มือถือ แท็บเล็ต คอมพิวเตอร์ หรืออุปกรณ์ใดก็ได้** ขอแค่มีเว็บเบราว์เซอร์และอินเทอร์เน็ต
* ข้อมูลทั้งหมดจะเก็บและเชื่อมต่อกับ Google Sheets บนคลาวด์ของคุณเสมอ ไม่ว่าจะเปิดใช้งานจากเครื่องไหนก็ตาม
